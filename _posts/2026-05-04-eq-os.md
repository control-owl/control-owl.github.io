# Introduction

After some heavy brain debugging, I dropped all ideas and now I focus on a new project eQ-OS. This should be a bootable minimal linux which when booted will start eQ app in kiosk mode.

Also it will have encrypted partition for saving wallet file. I will try to implement eQ update, since root is immutable. 

I managed to build it on my laptop and it is working fine. Right now I am trying to automate it with GitHub workflow but I am getting so many errors with mounting system and producing bootable image. 

Again I tried with fucking Docker. Problem is, I am using Arch Linux, my image is Debian.

I have a deja-vu.

Didn't I tried to play with Docker last summer? Oh man, here we go again. 

I hope this time it will not take so much time :D


And... then I realized that USB sticks are not a perfect solution for long term storage... Now I am researching M-Disk and a way to generate wallet and then burn it to a disk... This will take time.

But anyways, I will finish this some way or another.
